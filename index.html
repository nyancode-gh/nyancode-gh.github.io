About me

藤田雅幸ふじた まさゆき

1988年生まれ。滋賀県出身、岐阜在住。プログラミングやデザインが好きです。現在は、IT・WEB業界で働くことを目標にスキルを磨いたりお仕事を探したりしています。将来的には、関わってくださる全てのみなさまの成功をお手伝いできるような一人の人間として成長し、末長く社会に貢献し続けられる働き方を目指します。

株式会社YKA - Webサイト

Design / HTML / CSS / JS(JQuery) / PHP(独自CMSの設定)
