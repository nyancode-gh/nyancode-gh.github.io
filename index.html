About me

藤田雅幸 ふじたまさゆき

1988年生まれ。滋賀県出身、岐阜在住。プログラミングやデザインが好きです。現在は、IT・WEB業界で働くことを目指して、日々スキルを磨いたり、お仕事を探したりしています。将来的には、関わってくださる全てのみなさまの成功をお手伝いできるような一人の人間として成長し、末長く社会に貢献し続けられる働き方を目指します。

Works

株式会社YKA - Webサイト

Design / HTML / CSS / JS(JQuery)

職業訓練校のWebデザイナー科の制作実習で、ホームページリニューアルのお手伝いをさせて頂きました。4人のチームで、私は主にトップページのコーディング、レスポンシブ、デザイン、バナーの制作を担当しました。

介護事業を営む企業様でしたので、実際にサービスを受けるご家族やご利用者様が、見やすくて使いやすいと思ってもらえるようなデザインを意識し、コーディングにおいてはHTML5の規約に基づき、チームメンバーに伝わりやすい適切でシンプルなマークアップを心掛けました。

猫標識 - BLOG

Design / Word Press / CSS

にゃんこーど様（私）のブログをWordPressで制作致しました。サーバー・ドメインの契約から、必要なプラグインの導入、広告やアクセス解析等の各種設定、テーマのカスタマイズを担当しました。

デザインは、イメージカラーである黒をベースに、明朝フォントで高級感を演出し、カラム数やコンテンツの幅などCSSにて調整を行い、細かなニーズにも対応致しました。WordPressの技術（なかみ）を知るために開設したブログなので、今後は学んだことなどを発信するメディアとしても活用していきたいと思っています。

URL：https://nyancode.blog
